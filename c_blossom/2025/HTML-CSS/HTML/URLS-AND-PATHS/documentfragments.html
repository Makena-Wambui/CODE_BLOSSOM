Mailing Address

You might want to link to a specific part of a HTML document, known as a document fragment, instead of just to the top of the document. First assign an id attribute to the element you want to link to. Makes more sense to link to a specific heading or paragraph in a long document, rather than just to the top of the document.

Then to link to that specific id, include it at the end of the URL, preceded by a pound symbol.
