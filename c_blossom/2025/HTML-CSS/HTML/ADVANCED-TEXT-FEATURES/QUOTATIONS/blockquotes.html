If a section of block level content such as a paragraph, multiple paragraphs, or a list is quoted from somewhere else, wrap it inside a <blockquote> element to signify this. Then include a URL pointing to the source of the quote inside a cite attribute.

The HTML <blockquote> Element (or HTML Block Quotation Element) indicates the enclosed text is an extended quotation.
