In order to put an image on a web page, we use the <img> element.

It is a void element - it can not have any child content or have a closing tag.

It has two attributes: src and alt.

src attribute contains a URL pointing to the image we want to embed in our page.
The URL can be an absolute URL or a relative URL.
Without src attribute, <img> has no image to load.

Search engines also read image filenames and count them towards SEO.
Give your image a descriptive filename: dinosaur.jpg better than image.jpg

You could emebed the an image using it sabsolute URL.

Elements like img and video are sometimes called replaced elements.
This is because the element's content and size is defined by an external resource such as an image or video file,
not by the contents of the element itself.
