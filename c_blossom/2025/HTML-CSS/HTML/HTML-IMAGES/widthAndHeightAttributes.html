You can use width and height attributes of the img element
to specify the width and height of your image.
Provide their values as integers without a unit;
They represent the image's width and height in pixels.

Why do we need to specify width and height?
The HTML for your page and the image are separate resources that are fetched by the browser as separate HTTPS requests.
The browser starts to display the HTML to the user as soon as it receives it.
If the images have not yet been received, which is often the case,
as image files are often larger than HTML files,
the browser will only render the HTML, and will update the page with the image once the image is received.

Once the image is loaded, the browser adds the image to the page.
The image takes up space, so the browser has to move the text down the page
to accomodate the image above it.

Moving the text like this is extremely distracting, especially if the user had already started reading.

If you specify the size of the image in your HTML,
using width and height attributes, then the browser knows,before it even downloads it,
the space it has to allow for it.
So when the image is downloaded, the browser does not have to move the surrounding content.
